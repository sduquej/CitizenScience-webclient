webClientApp

Main module of the provided sample application

Routing is configured for the Main and List views.

Formly configuration is initially done registering the custom fields: fileField, currentLocation and mapLocation. This is done calling the formlyConfigProvider.setType() method. In addition, a templateManipulator is used to set the max property of inputs of type date to the current date. If it not intended to impose this upper limit, then comment or remove the code that adds the max property.
