restAPI
service in module webClientApp

Description

This service handles all the interaction with the back-end, REST application server. It exposes two methods, one for submitting and one for retrieving data; they both hit the endpoint configured here.

It uses Angular's $http service and the open source ng-file-upload directive.

Properties

  • base

    String with the base url of the backend REST server. If using a local instance it is okay to use localhost, however it is recommended to use the local IP address to test from external devices (that are on the same network).

    Example

    For a local backend server (deployed on the same machine)

    // local instance on port 9804
var base = 'http://localhost:9804';

    Using the intranet IP

    // suggested use, with the intranet IP
var base = 'http://192.168.1.35:9804';

    Remote REST server

    // url with hostname
var base = 'https://your.domain.com';

  • list

    This method hits the /api/v1/contributions/list endpoint of the base url with the GET method. If a file was submitted, the progress of the upload is updated on the formData object.

    Returns
    Array

    Data returned from the backend or an empty array if there was an error.

  • post

    This method hits the /api/v1/contributions/register endpoint of the base url with the POST method. If a file was submitted, the progress of the upload is updated on the formData object.

    Returns
    HttpPromise

    A promise with the result of the upload
