ListController
controller in module webClientApp

Description

Controller for the list/statistics view.

It uses the formConfig service to obtain the fields that make each contribution. The restAPI service is used to fetch the contributions data from the backend.

This controller manipulates the data and exposes what is intended to be shown on the List view.

Properties

  • contributions

    Array of contributions in descending order by date of contribution.

  • contributions

    Array of contributions in the last month.

  • contributions

    Array of the last ten contributions.

  • formConfig

    formParameters object, it's injected from the formConfig service.
